# REST Word Counter

Counts occurrence of words on a text through a REST Service.

Created for Feevale University Distributed Systems Course on year 2015.

## Getting started

```console
nvm use
npm install
PORT=3000 DEBUG='rest-word-counter:*' npm start
```

Access `http://localhost:3000`.

## Testing

```console
npm test
```
